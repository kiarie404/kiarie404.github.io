Hobo Kernel

  • Docs
  • Codebase
  • Releases
  • Demo

Hobo OS is a Kernel that is built to make installing and executing WASM modules in microcontrollers to be convenient.
Wasm modules in IoT promise better code portability and a much better capability-based security system than C or Docker packages. But this doesn't mean that wasm modules are Perfect... Haha.
This kernel exports the System calls using WASI prototypes and runs the wasm modules in Supervisor-mode. More explanations about the "why" can be found in the Documentation.
The kernel has been written in Rust and Riscv Assembly, targeting the Qemu-Riscv-virtual Machine.

About

He makes small kernels and writes drivers. Actively interested in embedded development. Mostly works with Riscv, but can adapt to other chips. So if you need to port your systems to Riscv, call this guy!! He's okay working in no-std environments, including debugging and testing. Quite a Chad

Obsessed with over-documentation. He can even over-document a Hello-world program, down to how the standard library of that hello-world works. With system designs and all. kidding.
